## 当前 `/graphs` 页面实现态摘要

- 主区上方为**总图谱**。
- 主区下方为**局部子图**与**审阅与策展卡**。
- 右侧常驻辅区为**整合后的紧凑上下文 + 根因列表**。
- 不再保留独立的**图谱提示**卡。
- 总图支持：
  - 点击空白取消当前总图选择；
  - 再次点击当前已选节点取消当前总图选择；
  - 点击另一个候选节点时同步切换右侧根因；
  - 点击非候选节点时清空右侧根因高亮，但保留图谱邻域上下文。
